Multi-Agent Early Urban Risk Detection System — Architecture Diagram

1️⃣
Upload CSV
User uploads complaints.csv via Streamlit file uploader
2️⃣
Extract
Extractor categorises each complaint by keyword matching + location tagging
3️⃣
Detect Pattern
Pattern agent applies threshold rules to identify risk type
4️⃣
AI Reasoning
Reasoning agent calls Gemini 2.0 Flash with a domain-specific prompt
5️⃣
Suggest Action
Action agent returns a rule-based municipal recommendation
6️⃣
Display Results
Streamlit renders alert, signals, reasoning and action to the user
