# CivicPulse AI

## Problem
Cities detect health and infrastructure risks too late.

## Solution
CivicPulse AI is a domain-specific AI agent that detects early risks from citizen complaints.

## Features
- Multi-agent system
- AI reasoning using Gemini
- Actionable alerts

## Impact
Detects risks early and reduces emergency response costs.

## How to Run
pip install -r requirements.txt  
streamlit run app.py
